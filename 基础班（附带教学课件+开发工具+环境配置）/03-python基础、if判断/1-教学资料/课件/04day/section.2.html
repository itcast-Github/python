if判断语句

<1>if判断语句介绍

  • if语句是用来进行判断的，其使用格式如下：
    if 要判断的条件:
        条件成立时，要做的事情
  • demo1:
    age = 30

    print "------if判断开始------"

    if age>=18:
        print "我已经成年了"

    print "------if判断结束------"
  • 运行结果:
    ------if判断开始------
    我已经成年了
    ------if判断结束------
  • demo2:
    age = 16

    print "------if判断开始------"

    if age>=18:
        print "我已经成年了"

    print "------if判断结束------"
  • 运行结果:
    ------if判断开始------
    ------if判断结束------

小总结：

  • 以上2个demo仅仅是age变量的值不一样，结果却不同；能够看得出if判断语句的作用：就是当满足一定条件时才会执行那块代码，否则就不执行那块代码

注意：

  • 代码的缩进为一个tab键，或者4个空格
